GIF
Cache
is a service to store and categorize GIF urls.
Since GIF files are often enormous, GifCache does not actually store the file itself. You enter the GIF's URL, label it if you want, and add some tags to categorize it. A small thumbnail is generated from the URL and displayed initially on a user's profile. The full GIF is only loaded when the thumbnail element is focused on.
Be aware that when using GifCache and viewing numerous GIFs, bandwidth usage can easily hit hundreds of megabytes very quickly.
